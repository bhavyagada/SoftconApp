# SoftconApp
Simple CRUD app using MEAN stack (MongoDB, Express.JS, Angular, Node.JS)

## Installation
To be done in both backend and frontend directories
```
npm install
```

## Run Backend
Node.js **v12.18.1** Mongodb **v4.4.2**
```
npm run serve
```

## Run Frontend
Angular **v11.0.3**
```
ng serve
```
